# ポイ活ツール類
## ポイ活に便利なツールをまとめています

### 各種ネット銀行の連続自動振込
詳しくは[ネコ山さん](https://www.youtube.com/@nekoyama000)の銀行ぐるぐるポイ活動画を参考にしてください。<br>
このツールはそこで紹介されていたネコ山氏作成の自動化スクリプトをPythonに移植し、より柔軟かつ簡単に使えるようにしたものです。<br>

## このプロジェクトには以下の機能が含まれています
- auじぶん銀行の連続自動振込
- SBI新生銀行の連続自動振込（未実装ですが、すぐに実装します。）
- その他
   
## このプロジェクトには以下の技術が使われています
- undetected-chromedriver: 銀行の自動化をするため、必須ではない可能性がありますがステルス性の高いWebdriverを使用します。

## インストール
1. `pip install -r requirements.txt`にて依存関係のインストールを済ませます
2. `config.yaml`ファイルを作成し、config_example.yamlの内容を参考に、設定を書き込みます
3. これで完了です。

## 使い方
`python3 main.py`にてメインメニューが開くため、そこから操作してください
